# Situation Report

Proactive assessment of current state — what needs attention, what actions are recommended.

## What Success Looks Like

The user gets a prioritized action list covering everything that matters right now. Nothing falls through the cracks. The report distinguishes urgent (act now) from important (act soon) from informational (be aware).

## Assessment Areas

Scan across all domains and surface what needs attention:

- **Open positions**: any past their freshness window (10-15 day decay timer)? Any with weakening thesis signals? Distance to stops?
- **Regime changes**: has the regime shifted since the last check? Does the current flag require portfolio adjustments?
- **Watchlist readiness**: any staged stocks approaching clearance conditions (breakout level, IV threshold, regime alignment)?
- **Portfolio health**: current heat, correlation exposure, cash reserve vs. target, sector concentration
- **Performance signals**: any strategy's rolling Sharpe declining? Any signal source accuracy dropping?
- **Wiki health**: last lint date, any stale pages flagged, any post-mortems pending for closed trades
- **Pending actions**: trades closed but not reviewed, synthesis overdue, regime snapshots needed

## Output

A prioritized list grouped by urgency:
- **Act Now**: positions at risk, regime changes requiring response, abort criteria approaching
- **Act Soon**: reviews pending, thesis decay alerts, watchlist opportunities nearing trigger
- **Be Aware**: performance trends, wiki maintenance, upcoming calendar events (earnings, OpEx, FOMC)

Keep it concise — one line per item with the key data point. Offer to drill into any item.
